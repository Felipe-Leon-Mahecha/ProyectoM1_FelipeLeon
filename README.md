# THE GEM — Landing Page

Landing page oficial de **THE GEM**, un juego arcade 2D hecho en JavaScript puro + HTML5 Canvas.

## Descripción

Esta página presenta el juego THE GEM, permite conocer sus características principales y enlaza directamente a la versión jugable en Netlify. Incluye una sección interactiva que demuestra la aleatoriedad del juego.

## Contenido del proyecto

- Página web semántica completa con header, nav, main, article, aside y footer
- CSS con variables HSL, Grid y Flexbox para layout responsive
- JavaScript con manipulación del DOM, eventos y generación aleatoria
- Documentación sobre HSL en los comentarios del CSS

## Cómo usar

Abre `index.html` directamente en tu navegador. No requiere servidor ni instalación.
